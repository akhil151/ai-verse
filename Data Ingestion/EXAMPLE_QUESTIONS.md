## Tips for Better Results

1. **Be Specific**: Instead of "funding", ask "What are the government funding schemes for tech startups?"

2. **Ask Follow-up Questions**: After getting an answer, ask for more details like "Tell me more about the application process"

3. **Use Natural Language**: The system understands conversational questions, so ask naturally

4. **Rephrase if Needed**: If you don't get good results, try rephrasing the question

5. **Check Your Data**: Make sure you've:
   - Run option 1 (Ingest PDFs) to process your documents
   - Run option 2 (Build Vector Database) to create the search index
   - Optionally run option 4 (RAG Assistant) for AI-powered answers
